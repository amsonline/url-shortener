Installation Guide

Step 1: Prepare the Server

Make sure your server meets the minimum requirements for running Laravel:

  • PHP >= 7.4
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP Extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Also, obviously, you need a domain set up on your server to use the features of this script. The preparation of the server depends on the server's OS and the Webserver.

There is a good documentation for this on Laravel website which you can read it here.

Also, if you are using Nginx as your web server, I have provided a sample conf file beside this documentation.

Step 2: Unzip the Package File

Unzip the package file containing your Laravel application into your server domain root.

Step 3: Install Composer Dependencies

Navigate to your Laravel project directory using Terminal and install Composer dependencies:

cd yourlaravelapp
composer install

Step 4: Run the installer

In order to prepare everything else and seed the initial data, you need to run the installer. To do so, navigate to https://your-website.com/install.php

The installer checks some requirements, gets some information from you ans installs the script completely.

Step 5: Navigate to admin area

After installation completes, it will give you the link to your admin area. It is https://your-website.com/admin. You have to enter your admin email and password defined in the installer and log into the admin area
